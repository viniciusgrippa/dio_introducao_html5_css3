Introdução ao HTML5 e CSS3

HTML5
    Versão mais nova do padrão HTML
    
    HTML é o padrão utilizado para criação de páginas WEB
    
    No HTML5 foram introduzidos novos elementos que proporcionam suporte a maior interatividade com o usuário.


CSS3
    Cascading Style Sheets (Folha de Estilo em Cascata)

    Utilizado para adicionar estilo / aparência a um documento WEB

    O CSS3 trouxe novas funcionalidades que permitem uma interação maior com o usuário, como uso de transições e animações.
